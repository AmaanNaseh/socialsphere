## Setup Guide:

1. Clone the repository

```bash
git clone https://github.com/xodosop1/socialsphere.git
```

2. Navigate to project folder, Install dependencies & run the project

```bash
cd socialsphere
```

```bash
npm install
```

```bash
npx nodemon server.js
```

3. Access website at : http://localhost:5000
